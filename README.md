## 📌 Projects

### 🖼️ BMP Viewer (C + SDL2)
A simple BMP image viewer written in C.

- Supports multiple BMP formats (Windows & OS/2)
- Handles RLE compression (RLE4, RLE8)
- Supports palette-based and bitfield images
- Uses SDL2 for rendering
