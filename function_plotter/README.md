# 📈 Function Plotter (C + SDL2)

A simple function plotter written in C using SDL2 and tinyexpr.  
It takes a mathematical expression as input and renders its graph in a 2D coordinate plane.

---

## ✨ Features

- Plots functions like `sin(x)`, `x^2`, etc.  
- Uses tinyexpr for expression parsing  
- Displays a grid and coordinate axes  
- Real-time rendering using SDL2  

---

## 🛠️ Requirements

- C compiler (GCC / Clang)  
- SDL2  
- tinyexpr (`tinyexpr.c` included)
